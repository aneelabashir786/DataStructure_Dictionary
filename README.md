##Chaining Hashing :

In this project I have used chaining hashing .	The whole dictionary has been loaded by chaining hashing . 

Chaining is a technique used forÂ avoiding collisions in hash tables. A collision occurs when two keys are hashed to the same index in a hash table. Collisions are a problem because every slot in a hash table is supposed to store a single element. Chaining hashing ends up collision by making linked list . It make it fast to access any element
